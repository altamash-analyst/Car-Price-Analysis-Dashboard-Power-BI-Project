# Car Price Analysis Dashboard | Power BI Project

A professional Business Intelligence project built using Microsoft Power BI, DAX, and Excel-based data modeling to analyze car pricing trends and automobile market insights.

This project transforms raw car data into an interactive analytics dashboard that provides insights into vehicle pricing, brands, models, specifications, and market trends.

---

## Project Overview

The objective of this project is to develop an end-to-end data analytics solution using real-world automobile data.

This project includes:

- Data cleaning and preparation
- Data transformation
- Data modeling
- DAX measure creation
- Interactive Power BI dashboard development
- Business insight generation

The dashboard helps users understand car market trends, pricing behavior, and factors influencing vehicle prices.
